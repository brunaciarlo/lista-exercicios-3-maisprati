<h2 id="comecando">🚀 Começando</h2>

Para testar os exercícios em sua máquina, é necessário seguir os passos abaixo:

<h3>Clonando</h3>

Como clonar

Com a pasta desejada selecionada no terminal, execute o comando: 

```bash
git clone https://github.com/brunaciarlo/lista-exercicios-3-maisprati.git
```

<h3>VS Code</h3>

Agora você pode executar os exercícios de duas maneiras:

Certifique-se que possui a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

Abra a pasta com todos os exercícios no editor de código VS Code, clique com o botão direito no arquivo .html que deseja abrir e selecione:
```bash
Open with Live Server
```

<h3>Navegador</h3>

Navegue até a pasta do projeto no seu computador.

Dê dois cliques no arquivo .html.

Ele será aberto no seu navegador padrão.

<h3>Exercícios</h3>

| Arquivo  | Exercícios |
| ------------- | ------------- |
| index.html  | Estrutura usando tags semânticas h1, p, main, h2, footer |
| listas.html  | Lista não ordenada com cinco hobbies, lista ordenada de receita, nav com links externos |
| feedback.html  | Formulário com nome, e-mail, endereço, tipo de feedback, campo para o feedback e botão de envio  |
| galeira.html  | Galeria com oito imagens e legendas organizadas por grid + menu hamburguer  |
| styles.css  | CSS básico - linkado ao index.html - aplicações de background-color, color, padding, margin, border... |
| style-galeria.css  | CSS linkado ao galeria.html - ajuste de grid, flexbox, uso de media query e configurações do menu hamburguer |
